What's this app for?

The EPA GeoPlatform Dashboard provides information on GeoPlatform items. It allows the user to audit and edit this information in order to comply with EPA guidelines.

This guide will walk you through using the Dashboard. These instructions will step through the navigation menu on the left side of the screen beginning with GeoPlatform Online (GPO) Content. More content is currently being added to this application and help documentation will continually be added as more functionality is added.

Signing In/Out

The navigation bar in the top right corner of the application has two buttons, Sign In/Sign Out, and Help. Clicking the Sign In button will prompt the user for sign-in credentials for the GeoPlatform Online (GPO). Either Esri-based GPO credentials or your EPA Enterprise Single Sign-On credentials can be used. Once you have signed in, the sign in button will display your user name and your GPO Role (eg. User, Admin, etc.) next to the Sign Out button.

Initial View

The initial view displays a few key items across the top of the screen above the data table (Public Items Audit Result, My Items, My Groups, Auth Groups). 'Public Items Audit Result' displays the percentage of public items that have passed the Audit. The audit is run throughout the life cycle of the application, so this number will reflect what is current. 'My Items' displays the number of Items associated with your GPO account. 'My Groups' displays the number of groups associated with your account. 'Auth Groups' display the number of Authoritative groups that your account is associated with. The names of those groups can be displayed when the number is clicked. When clicked, either the authoritative group you are associated with will display or, if your GPO account is associated with multiple authoritative groups, then a dropdown will display with a list of the Authoritative groups. By choosing one of these groups it will refresh the table below with the content from the chosen group.

GeoPlatform Online Content

GeoPlatform Online Content Table

The GPO Content table displays the GPO items. The columns of the table are defined below:

Title The title of the item. This is the name that's displayed to users and by which they refer to the item. Every item must have a title.
Access Indicates the level of access to this item: private, shared, org or public.
Type The GIS content type of this item. Examples include Web Map, Map Service, Shapefile, and Web Mapping Application. To see all the available types see ArcGIS Resources.
Views Number of views of the item.
Owner The username of the user who owns this item.
Audit This field indicates if the item pass for failed the audit.

Searching and Filtering Table

Searching - You can search the GPO content table by entering a word or phrase to search by in the search text box in the top right corner of the table. This will search all items and columns.

Filter - You can filter what is displayed in the GPO content table by clicking on the dropdown menus under the title for each column. For the title column you must type in text to filter the table.

Item Details

The item details for a GPO item in the table can be displayed by clicking on the row. The item details display the following information for an item.

Title The title of the item.
Summary a short summary description of the item.
Description Item description.
Credits Any license information or restrictions.
Access and use Constraints Information on the source of the item and its copyright status.
URL The URL for the resource represented by the item. Applies only to items that represent web-accessible resources such as map services.
Thumbnail The URL to the thumbnail used for the item.
Tags User defined tags that describe the item.

Editing Item Details

Editing Item Details - An items details can be edited by changing the value in the textbox next to the item detail that you wish to change.

Changing Item Thumbnail - The thumbnail for an item can also be changed by clicking the change button under the thumbnail preview and navigating to the image that you would like to use as the thumbnail. For best results, the image should be 200 pixels wide by 133 pixels high. Acceptable image formats are: PNG, GIF and JPEG.

Adding/Removing Tags - Tags can be added and removed from an item. To add a tag an item type the tag that you would like in the text box under the Tags title, then click the Add button. The tage will be added to the list below the Add button. To remove a tag, select the tag in the Tags List and click the Remove button.

User Management

This page allows administrators to manage the sponsorship of external (non-single-sign-on) accounts. Every external account must have a sponsor, and accounts must be reviewed and renewed every 90 days, or the account will be automatically disabled. Disabled accounts will be cleaned up periodically with their content items transferred to the most recent sponsor. The table shows the status of each account, when their account was last renewed, and when they are due for renewal. Any administrator may sponsor or renew the sponsorship of any external account.

When sponsoring an external account, please select a reason for the sponsorship from the dropdown, enter the name of the external account holder's organization and an optional short description of the context, and select an authoritative group that member should be a part of for auditing purposes. If an account is subsequently sponsored by a different group, they will remain a member of all sponsoring authoritative groups until the account is disabled or they are manually removed from a group.
